# Tidus Documentation

> The AI system that governs all AIs.

## What is Tidus?

Tidus is an enterprise AI governance and routing system. Every AI request in your organisation passes through Tidus, which selects the cheapest capable model, enforces budgets, and prevents runaway agent costs.
